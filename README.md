# 주가 분석 대시보드 📈

여러 주식 종목의 수익률을 비교하고 기술적 분석을 수행할 수 있는 웹 애플리케이션입니다.

## 주요 기능

- 📊 **수익률 비교**: 여러 종목의 누적 수익률을 한 눈에 비교
- 🕯️ **캔들스틱 차트**: 상세한 가격 움직임 확인
- 📈 **이동평균선 (MA)**: 5일, 20일, 60일 등 다양한 기간
- 📉 **볼린저 밴드**: 변동성 기반 매매 신호
- 🔄 **RSI 지표**: 과매수/과매도 구간 확인
- 🔗 **상관관계 분석**: 종목 간 연관성 파악
- 📊 **거래량 분석**: 거래 활성도 비교

## 지원 시장

- 🇺🇸 **미국 주식**: AAPL, MSFT, TSLA 등
- 🇰🇷 **한국 주식**: 삼성전자(005930.KS), 네이버(035420.KS) 등

## 사용법

1. 왼쪽 사이드바에서 티커 입력 또는 빠른 선택 버튼 클릭
2. 분석 기간 설정
3. 분석 옵션 선택 (이동평균선, 볼린저 밴드, RSI 등)
4. "분석 시작" 버튼 클릭

## 기술 스택

- Python 3.10+
- Streamlit
- yfinance
- Plotly
- Pandas
